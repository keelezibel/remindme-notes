## Stack

- Operations: insert, peek, remove
- Used in recursion, DFS

## Queue
- FIFO
- Operations: insert, peek, remove
- Implementation: collections.deque
- Operations: insert front, insert back, remove front, remove back, peek front, peek back
- append(), appendleft(), popleft(), pop(), len()
